To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need **nodejs**. To download **nodejs** visit [nodejs.org](https://nodejs.org/en/download/).

```sh
   npm install
   node filename.js
```


### Setup

```sh

   cd Data-Structures-and-Algorithms
```
